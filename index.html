üöÄ Quick Start

1. Installation

Clone the repository and install dependencies:

git clone https://github.com/luizf1lho/brp-portfolio-optimizer.git
cd brp-portfolio-optimizer
pip install -r requirements.txt

2. Run the Application

Start the Streamlit web interface:

The application will open at http://localhost:8501

3. How to Use

  1. Upload Tab: Upload your CSV file with trading data
  2. Results Tab: View optimized metrics and allocation
  3. Report Tab: Generate and download professional reports
  4. QuantStats Tab: Analyze performance vs benchmark

üìä Algorithm Details

Markowitz Mean-Variance Optimization

The optimizer uses the classic Markowitz approach to find the optimal portfolio weights that:

  • Maximize expected returns (Sharpe ratio)
  • Minimize portfolio volatility
  • Respect risk constraints (max drawdown tolerance)
  • Apply position sizing multipliers for each strategy

Key Metrics Calculated

  • Sharpe Ratio: Risk-adjusted return metric
  • Sortino Ratio: Similar to Sharpe, penalizes downside volatility only
  • Calmar Ratio: Return to maximum drawdown ratio
  • Maximum Drawdown: Peak-to-trough decline
  • Win Rate: Percentage of profitable trades
  • Profit Factor: Ratio of gross profit to gross loss
üìå Note: Uses 252 business days/year for annualization

  • Strategy name (Global): Identifier of the strategy
  • Profit/Loss (Global): Trade result in currency units
  • Size: Position size or lot
